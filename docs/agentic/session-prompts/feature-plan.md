# Feature Plan Launcher

Use this launcher to write or refresh a serious Lineup Desktop feature/design
plan.

## Read Order

1. [`AGENTS.md`](../../../AGENTS.md)
2. [`docs/AGENTIC_DEV_WORKFLOW.md`](../../AGENTIC_DEV_WORKFLOW.md)
3. [`docs/agentic/plan-authoring-standard.md`](../plan-authoring-standard.md)
4. [`docs/architecture/CURRENT_STATE.md`](../../architecture/CURRENT_STATE.md)
5. task-specific docs or source files

## Output Requirements

- Declare `**Task family:** feature/design`.
- Name the architecture seam.
- Name files in and out of scope.
- Name verification commands and expected outcomes.
- Name replan triggers.
- Record import ledger obligations when copied/adapted upstream code is in
  scope.
