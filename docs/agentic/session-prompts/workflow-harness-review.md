# Workflow Harness Review Launcher

Use this launcher for read-only review of Lineup Desktop workflow, verifier,
launcher, project skill, role, or control-plane behavior.

## Review Surface

- `AGENTS.md`
- `docs/AGENTIC_DEV_WORKFLOW.md`
- `docs/agentic/`
- `.agents/skills/`
- `tools/verify-docs.mjs`
- `tools/verify-redaction.mjs`
- package scripts

## Required Checks

- Workflow routes desktop work as feature/design.
- Tier 3 work uses the desktop feature-quality loop.
- Historical upstream cleanup program mechanics are not active authority.
- Launcher project skills are thin wrappers over tracked launcher docs.
- Same-name workflow and boundary skill adaptations stay concise, read tracked
  docs first, and do not become a second workflow source of truth.
- Plan standards require evidence, scope, seam decisions, verification
  classification, acceptance criteria, rollback, and replan triggers without
  forcing irrelevant sections.
- Verifiers fail on missing required docs, broken links, baggage files, and
  redaction leaks.
- Verifiers fail when required workflow anchors or project skills drift.
- Launcher docs require review before closeout.
- Verification scripts are real checks, not placeholders.

Lead with findings ordered by severity. Say explicitly when no blockers remain.
