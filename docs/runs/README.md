# Runs

Use this directory for local run bundles when a task is too large for one
session and needs repeated handoff. Raw run bundles are intentionally ignored by
git; promote durable conclusions into tracked docs when they need to survive.

Use `docs/runs/archive/plans/` for local copies of completed full plan bodies
after closeout. This archive is intentionally ignored and is not repo
authority; durable lessons belong in tracked roadmap, architecture, workflow,
import-ledger, or verifier docs.

Do not commit raw logs, media samples, crash dumps, diagnostics containing
secrets, or local-only scratch output here.
